MPIFX
Modern Fortran wrappers around MPI routines

The open source library MPIFX is an effort to provide modern Fortran (Fortran 2003) wrappers around routines of the MPI library to make their use as simple as possible. For more information see the following sources:

  • Online documentation for installation and usage of the library
  • API documentation for the reference manual.
